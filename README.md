# Sendsar SDK for JavaScript

JavaScript client for Sendsar realtime chat. Use a **session JWT** from your backend — never `sk_*` in the browser.

## Install

```bash
npm install @sendsar/chat-sdk-javascript
```

## CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@sendsar/chat-sdk-javascript@0.6.1/dist/sendsar.browser.min.js"></script>
```
